# Hospital Management System

A MySQL-based relational database project for managing:

- Patients
- Doctors
- Appointments
- Medical History
- Doctor Schedules

## Features

- Appointment management
- Doctor scheduling
- Patient records
- Medical history tracking
- Relational database structure

## Tech Stack

- MySQL
- SQL
- phpMyAdmin
- MySQL Workbench

## Future Improvements

- React frontend
- Node.js backend
- Authentication
- Admin dashboard
