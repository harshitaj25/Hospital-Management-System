# Hospital Management System

A MySQL-based relational database project for managing:

- Patients
- Doctors
- Appointments
- Medical History
- Doctor Schedules

## Features

- Appointment management
- Doctor scheduling
- Patient records
- Medical history tracking
- Relational database structure

## Tech Stack

- MySQL
- SQL
- phpMyAdmin
- MySQL Workbench

## Contributions

Contributions, improvements, and suggestions are welcome.

If you'd like to add features, improve the project, or fix issues, feel free to fork the repository and submit a pull request.

## Future Scope

- React frontend
- Node.js backend
- Authentication system
- Admin dashboard
- Online appointment booking
- AI-based healthcare assistance
